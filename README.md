东南大学访客申请，SEU访客车辆入校预约，基于immortalwrt里的Shell可实现定时自动提交申请

只有校内学生可用，需要先在网页上自行申请一次，并抓包得到raw.json

代码中调用的raw.json是提交申请doAction请求的负载formData，由于涉及隐私信息，请自行抓包获取
